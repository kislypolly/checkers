# Проект "Шашки": многопользовательская онлайн-игра

Этот проект представляет собой реализацию классической игры в шашки с возможностью игры по сети и общением с соперником по чату. 

## Используемые технологии

Проект построен на базе Python и использует следующие ключевые технологии:

*   **Flask**
*   **Flask-SocketIO**
*   **WebSockets**
*   **HTML/CSS/JavaScript**

## Об игре

"Шашки" — это классическая настольная игра для двух игроков. Проект реализует базовую механику шашек, позволяя двум удаленным игрокам играть друг против друга в одной сессии.
Зайти на игру можно по ссылке: https://web-production-08cc2.up.railway.app/
