О проекте

Многопользовательская онлайн-игра в шашки: матчмейкинг 1 на 1, чат в реальном времени, опциональные аккаунты с кубками и email-уведомления при подключении соперника.

Авторизация

  • Регистрация опциональна — можно играть как гость или войти через «Аккаунт».
  • После входа сервер запоминает пользователя в cookie-сессии.
  • Пароли хранятся в виде хэша (Werkzeug), не в открытом виде.
  • Кубки начисляются только зарегистрированным игрокам.

Email-уведомления (SMTP)

  • Когда соперник подключается к очереди, зарегистрированному игроку уходит письмо на email из профиля.
  • Тема: «противник найден».
  • Отправка по протоколу SMTP (smtplib).
  • Шифрование соединения — STARTTLS (порт 587).

Используемые технологии

  • Python
  • Flask
  • Flask-SocketIO
  • WebSockets
  • SMTP
  • HTML / CSS / JavaScript

Об игре

«Шашки» — классическая настольная игра для двух игроков. Проект реализует базовую механику шашек, позволяя двум удалённым игрокам играть друг против друга в одной сессии.

Исходный код доступен на GitHub.
