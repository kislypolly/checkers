Информация о проекте

О проекте

Многопользовательская онлайн-игра в шашки с возможностью игры по сети и общением с соперником по чату.

Используемые технологии

  • Python
  • Flask
  • Flask-SocketIO
  • WebSockets
  • HTML / CSS / JavaScript

Об игре

«Шашки» — классическая настольная игра для двух игроков. Проект реализует базовую механику шашек, позволяя двум удалённым игрокам играть друг против друга в одной сессии.

Исходный код доступен на GitHub.
